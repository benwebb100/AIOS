# SOP — Sales Process

> Lead → discovery → proposal → setup → retainer. End-to-end flow for closing a client.
> Source: Master Playbook Part 3 + Appendix B + Appendix C.

## 1. Lead
- Source: Meta ad → landing page → Calendly booking (primary), or referral
- Calendly auto-confirms; calendar event drops into `03 Meetings/`
- Add to `Clients.md` Prospects table at stage `discovery booked`

## 2. Discovery Call
See `Discovery-Call.md`. 30 minutes.

## 3. Proposal (same day as discovery)

Populate this template from the discovery transcript:

```
PROPOSAL FOR: [Business Name]
DATE: [Date]
PREPARED BY: Bailey — AIOS Agency

WHAT I HEARD
- [Pain point 1]
- [Pain point 2]
- [Pain point 3]

WHAT I'D BUILD
Phase 1 — Setup (Week 1):
- Full business context loaded into AI operating system
- Connected tools: [list]
- First automation: [targeting their #1 pain point]

Phase 2 — Ongoing (Monthly):
- Daily business brief
- [Automation 2]
- [Automation 3]
- Monthly strategy call + async support

INVESTMENT
- Setup: $[X] one-time
- Monthly: $[X]/month
- Founding pricing — locked for 6 months (if applicable)

NEXT STEPS
1. Sign off
2. Share tool logins / API keys (checklist sent)
3. Schedule setup session (2–3 hrs, remote)
4. AIOS live within 48 hours
```

## 4. Close
- Send via email + calendar follow-up 24 hrs later if no reply
- Common objections + scripts: see Master Playbook Appendix C

## 5. Setup Session
- 2–4 hrs, remote (or in-person for premium tier)
- Install Layers 0–2 (foundation, context, data) + first automation
- Full step-by-step: Master Playbook Part 3 / Phase 2
- Update `Clients.md` — move from `proposal sent` → `closed-won`, then to **Active Clients** table

## 6. First Week
- Daily check-ins on the live automation (typically the daily brief)
- Day 7: 15-min check-in call, capture feedback
- Refine context, fix integrations

## 7. Retainer Kicks In
- 1–2 automations / month
- Context updates and maintenance
- Monthly 30-min strategy call
- Async support (24-hr response on business days)

## Scope Discipline
**In retainer:** 1–2 automations/month, context updates, monthly call, async support.

**NOT in retainer:** Custom software builds, marketing execution, 24/7 on-call, anything > 8–10 hrs/month per client.

If the ask is bigger, scope it as a separate project with a separate fee. Never let it creep.
