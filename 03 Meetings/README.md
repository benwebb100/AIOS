# 03 Meetings

Meeting notes, recordings, and transcripts. One file per meeting.

**Naming:** `YYYY-MM-DD - [Counterparty] - [Topic].md`
Examples:
- `2026-05-04 - Acme Pty Ltd - Discovery.md`
- `2026-05-11 - Ben - Weekly Sync.md`
- `2026-05-15 - Acme Pty Ltd - Setup Session.md`

Discovery calls, setup sessions, weekly Ben syncs, monthly retainer calls all land here. Transcripts can be pasted at the bottom; key takeaways and action items at the top.
